# vim-config

My Vim configuration file
F7: compile c++ file - F8\n
F2: debug c++ file - F6\n

gd: jump to definition - Ctrl-o jump back\n
,rn: rename function\n
,dd or F8: start debuging\n
,dx: exit debuging\n
